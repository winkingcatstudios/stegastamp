Stegastamp

A step-by-step guide using open source tools to insert author and license data into an image with steganography and timestamp a file to prove creation date.

Stegastamp a file

How it works

Stegastamp is a process to easily and quickly protect the authorship of images before posting them to the internet. The intention is to create a simple process by which content creators can embed their name and creative license into an image then timestamp the embedded file to authenticate its creation date.

STEGA-STAMP: 'stega' refers to the use of an opensource steganography library to embed words into an image file. "stamp" refers to the use of an open blockchain timestamping service. The use of free and open services aids in the transparency and authenticity of authorship proofs.

Other methods for proving authorship include file metadata and sending files through physical or electronic mail. These methods often elicit suspicion of tampering and can make it difficult to verify authoriship on a mass scale or anonymously in the case an author using an alias.

Steganography without timestamping: Changing or compressing an image can break steganography. This allows someone to use their own steganography after the author giving the author no way to prove that they were the original.

Timestamping without steganography: Sending an .ots timestamp file so others can verify the date allows anyone with access to the .ots file to claim they are the original author.

Steganography + Timestamping: Authors can send their original file and .ots file to anyone online. The .ots file verifies the date, and the original verifies the author. Subsequent steganography would result in a mismatch with the .ots file hash.

Creation and Proof Steps

Stegastamp Creation Steps
Step 1: Click "Choose File" to upload an image you created.
Step 2: Enter author name and license. Optionally, enter a password for private proofs.
For creative license suggestions, see Creative Commons Licenses
Step 3: Upload the newly saved image to OpenTimestamps.org and save the .ots file alongside your encrypted image.


Stegastamp Proof Steps
Step 1: Click "Choose File" to upload a stegastamped image.
Step 2: Confirm author and license. If nothing appears, a password is required.
Step 3: Upload .ots file to OpenTimestamps.org
Step 4: Upload the corresponding stegastamped image file to confirm timestamp date.

NOTE: It will take 1-2 hours before newly created .ots files can be verified.
"Pending attestation" indicated this delay.
